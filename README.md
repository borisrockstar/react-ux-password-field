react-webpack
=============

Template: React + webpack

### Setup
``` npm install webpack -g ```

``` npm install bower -g ```

``` npm install webpack-dev-server -g ```

### Run The DevServer
``` npm run devserver ``` -> will start a server at [http://localhost:9500/webpack-dev-server/](http://localhost:9500/webpack-dev-server/)


### Includes
- configuration files
- webpack
- facebooks react
- webpack-Loaders
    - JSX-Loader
    - Styles-Loader
    - CSS-Loader
    - LESS-Loader
    - Script-Loader
